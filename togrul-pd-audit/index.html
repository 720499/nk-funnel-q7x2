Канал · как устроен

ПроДокторов: рейтинг ≠ позиция, аукцион, машина отзывов

Чтобы аудит читался верно — короткая механика площадки (МедРокет). Важно различать два механизма: звёзды (рейтинг) и позицию в списке. Звёзды купить нельзя, позицию — косвенно можно. Всё ниже — из официального help.prodoctorov.ru; конкретные ставки на дату видны только в кабинете.

Рейтинг (звёзды 0–5)

Только из отзывов, купить нельзя. 50% клиника + 50% врачи. Влияют только авторизованные отзывы; подробный весит больше; со временем вес падает (~6 мес). У Art Dental — 4.5.

Позиция в общем списке = баллы 730

Балльное ранжирование клиники, максимум 730 баллов. Звёзды весят лишь ~20%; ~80% позиции «покупается» косвенно через платные опции и интеграцию МИС.

Из чего складываются 730 баллов клиникиофициально

В денежных списках по услугам (имплантация / брекеты / виниры) ранжирование идёт не по баллам, а по иерархии: топ-3 = только спецразмещение (аукцион) → онлайн-запись → телефон + цены → прочие. Сколько у Art Dental реально набрано из 730 — [ЛК-закрыто], по карточке виден только верхний слой (рейтинг есть, онлайн-запись есть, видеовизитки/Клуба нет).

Аукцион спецразмещения · GSP

Топ-3 в денежных списках — это аукцион, не фикс-прайсофициально

Главный платный продукт. Блок «Спецразмещение» — максимум 3 места, на 1 сутки (00:00–24:00 МСК), с пометкой по 347-ФЗ. Механика — GSP (аукцион второй цены): три высшие ставки берут места, победитель платит ставку следующего за ним.

  • ПодачаДо 23:00 МСК на завтра (день-в-день нельзя), есть галка «повторять ежедневно».
  • Рекоменд. ставка= вчерашняя ставка 1-го места +1% → площадка сама разгоняет цену.
  • Тай-брейкПри равных ставках выше встаёт тот, у кого выше рейтинг; карточка врача бьёт карточку клиники.
  • Стоимость записиОтдельно от ставки: 3,7% от стоимости услуги в коридоре min/max по городу, платится за состоявшуюся запись.

Почему здесь нет конкретных ставок по Самаре

Ставка спецразмещения — динамическая (аукцион), своя на каждый день, раздел и город, и видна только в кабинете на дату. У нас нет ЛК Art Dental → ставки по Самаре [ЛК-закрыто]. Цифры из чужих городов/планёрок сюда не переношу — они искажаются и устаревают. Принцип аукциона привожу, конкретные суммы — нет.

Машина отзывов · слова-ключи

Самый дешёвый рост — поток отзывовофициально

Звёзды купить нельзя — только набрать отзывами. Поток должен быть постоянным (вес отзыва падает за ~6 мес). Это работает даже без платного размещения.

«Сила отзыва» (легально)

QR-полиграфия в клинике → пациент сканирует → оставляет отзыв → получает промокод-скидку. За N промокодов клинике дают платное ПО бесплатно на время. На рейтинг влияют только авторизованные подробные отзывы; отвечать ≤24 ч.

Слова-ключи в отзывах = SEO-входы

ПД авто-линкует названия услуг и болезней в тексте отзывов → они ведут на городские списки по услуге. Каждый отзыв с услугой = дополнительная точка входа трафика. Управляешь этим, заполнив услуги и профиль лечения в карточке.

⚠️ Накрутка / заказ отзывов → бан (см. риски ниже). Машина отзывов = легальный сбор у реальных пациентов, а не покупка.

Трудности канала

Почему нельзя вешать загрузку только на ПДотрасль

Обратная сторона площадки — учитывать до того, как заливать в неё бюджет. Эти риски работают против любой клиники на ПД.

  • Карательная модерация

    Фейк-отзыв от конкурента → жалоба → обнуление рейтинга и блок: клиника на 3 года, врач на 1 год + баннер «покупала отзывы». Площадка платит жалобщику — это стимулирует доносы. Прецедент-суд в восстановлении рейтинга отказал. Откатить нельзя.

  • 30–35% отзывов отклоняются

    Даже реальные — по множеству причин. Теряются настоящие положительные отзывы → поток нужен с запасом.

  • Ловушка зависимости

    Когда доля записей через площадку переваливает за 40%, комиссию можно поднять, а отказаться уже нельзя — маржа падает. Красный флаг — агрегатор >40% записей.

  • Обесценивание бренда + профиль врача

    Логика «пациент выбирает врача» превращает клинику в «арендный центр». Нельзя удалить профиль своего врача (только сам врач через поддержку). На карточке крутится реклама конкурентов — убирается лишь «Ультимой».
